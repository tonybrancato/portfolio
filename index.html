My Projects

  • Good in the Hood
    •

Good in the Hood allows a user to find restaurants and nightlife in the neighborhood they're currently standing in. The app is an API hack utilizing the Google Maps API and the Zomato API. Utilizing the HTML5 Geolocation, the user is able to pinpoint their location and receive the most relevant results. The app also makes use of jQuery for DOM traversal.

  • GameKeeper
    •

GameKeeper allows you to keep track of what games you have and how often they're played, regardless of their medium. Utilizing Node.JS with the Express Framework and MongoDB with Mongoose, a user can now keep track of every game they own. For the front-end, I used jQuery for DOM manipulation and traversaland Velocity.js for animations. Finally, I used Mocha and Chai for testing.

  • Fit Pets
    •

Fit Pets can help you track your dogs health so you can ensure that they maintain a healthy weight, or are on track to doing so. I used React for the front-end along with Redux for maintaining state. On the backend, I used MongoDB and Mongoose, along with Node.js with the Express framework. Enzyme and Jest were used for unit testing in the React portion with Mocha and Chai being used on the backend.
